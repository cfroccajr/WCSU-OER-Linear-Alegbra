If an \(n\times n\) matrix \(A\) has \(n\) distinct eigenvector, \(\lambda_1,\ldots,\lambda_n\text{,}\) and corresponding eigenvectors, \(\vec{x}_1,\ldots,\vec{x}_n\text{,}\) then \(A\) is diagonalizable and in particular

\begin{equation*} A=PDP^{-1} \end{equation*}

where the columns of \(P\) are the eigenvectors,

\begin{equation*} P= \left[ \vec{x}_1\, \vec{x}_2\, \cdots\, \vec{x}_n \right], \end{equation*}

and \(D\) is a diagonal matrix with the eigenvalues on the diagonal,

\begin{equation*} D= \left[ \begin{array}{ccccc} \lambda_1 \amp 0 \amp 0 \amp \cdots \amp 0\\ 0 \amp \lambda_2 \amp 0 \amp \cdots \amp 0\\ \amp \amp \ddots \amp \amp \\ \amp \amp \amp \ddots \amp \\ 0 \amp 0 \amp 0 \amp \cdots \amp \lambda_n\\ \end{array} \right]. \end{equation*}
