Definition 3.1.4.

A function \(f:V\rightarrow W\) is one-to-one if for all \(v_1,v_2\in V\text{,}\) \(f(v_1)=f(v_2)\) implies \(v_1=v_2\text{.}\) The function is onto if for all \(w\in W\) there exists \(v\in V\) such that \(f(v)=w\text{.}\) A function which is both is called an isomorphism.
