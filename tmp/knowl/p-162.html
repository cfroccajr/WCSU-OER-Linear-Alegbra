Let \(f:\mathbb{R}\rightarrow\mathbb{R}\) be defined by \(f(x)=-7x\) and let's see that this is linear. First, since both the input and output for the function comes from \(\mathbb{R}\) that is the domain and the codomain. If \(x=ax_1+bx_2\text{,}\) then we get

\begin{align*} f(x)\amp = f\left(ax_1+bx_2\right)\amp\amp \mbox{definition of}\ x\\ \amp = -7\left(ax_1+bx_2\right)\amp\amp \mbox{definition of}\ f\\ \amp = -7ax_1-7bx_2 \amp\amp \mbox{distributive property}\\ \amp = a\left(-7x_1\right)+b\left(-7x_2\right) \amp\amp \mbox{commutative property}\\ \amp = af\left(x_1\right)+bf\left(x_2\right)\amp\amp \mbox{definition of}\ f \end{align*}

Therefore, \(f(x)\) satisfies the definition of a homomorphism (linear function).
