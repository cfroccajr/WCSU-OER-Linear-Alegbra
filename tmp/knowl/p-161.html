A function is a relation, call it \(A\text{,}\) from one set called the domain, \(X\text{,}\) to another set called the codomain, \(Y\) such that

  • \(\forall x\in X,\ \exists y\in Y:\ A(x)=y\)
  • \(\forall x\in X:\) if \(y_1=A(x)\) and \(y_2=A(x)\text{,}\) then \(y_1=y_2\text{.}\)

If in addition we know that \(A(ax+by)=a\, A(x)\, +\, b\, A(y)\) we say that the function is a homomorphism, or that it is a linear map.
