Investigation 3.1.4. Surprisingly Not Linear.

Let \(f:\mathbb{R}\rightarrow\mathbb{R}\) be defined by \(f(x)=x+1\) and let's see that this is not linear. Since we want to see that something is not the case we just need a counter example. If \(x=1\) and \(y=2\) then

\begin{equation*} f(x)+f(y)=f(1)+f(2)=2+3=5, \end{equation*}

but

\begin{equation*} f(x+y)=f(3)=4 \end{equation*}

so that \(f(x)+f(y)\neq f(x+y)\text{,}\) i.e. \(f\) is not linear. This may be surprising since \(f(x)=x+1\) is a line, but this is what is called an affine map since it is the sum of a linear map and a constant.
